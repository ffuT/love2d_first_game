# LÖVE 2D First Game

This is a learning project for me to learn LÖVE 2D and Lua in general

## Project Structure

- **main.lua**: The main entry point of the game
- **Entities.lua**: This file handles the different game entities, including their creation, behavior, and interactions
- **Physics.lua**: Contains the physics logic, including collision and body2d movements
- **Shaders/**: Directory that contains shaders for the project
